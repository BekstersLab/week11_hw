About

This is a simple website written in Python with the Flask micro web framework.

It has built upon last week's homework. Changes and improvements to the code include:

  • Splitting the single app file into separate files using Jinja templates to remove duplicated HTML code
  • Implementing Bootstrap 5
  • Creating a contact form that stores a user's first name, last name, email and message in a database
  • Changing the home page welcome message based on the time of day using the datetime module to determine the current time and an if statement to update the message based on the hour
  • Linking to Google Fonts to incorporate open-source fonts to improve the website's typography
  • Creating a dynamic portfolio page that goes to database to source the content and links.
